01/05 mission
The next breakthrough in AI for science is not better models.
It is better organization.

AI agents can now search literature, write code, analyze data, and design molecules. But complex scientific problems require teams of agents and humans coordinating across days or weeks. No existing tool provides this coordination.

We build open-source infrastructure that makes hybrid human-AI research auditable, reproducible, and inspectable. Our first product is WorkGraph, a task coordination graph where both humans and AI agents claim, execute, and complete interdependent work, with the full process recorded in a browsable computation graph.

PHR methodology comparison

A three-way comparison on pseudo-homologous region (PHR) discovery in a defined pangenome interval: WorkGraph with Erik in the loop alongside specialist agents, a single-session agentic CLI running autonomously, and a solo skilled bioinformatician using vg / PGGB. Each arm produces a candidate set; the WorkGraph arm produces an inspectable computation graph alongside it. writeup → coming soon
